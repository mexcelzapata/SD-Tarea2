## Instalación de Kafka 
inicialmente descargamos kafka 2.8.1

```
mkdir kafka
cd kafka
wget https://downloads.apache.org/kafka/2.8.0/kafka_2.13-2.8.0.tgz
tar -xzf kafka_2.13-2.8.0.tgz

```
